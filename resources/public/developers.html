Hbg-Crime.org API

There are currently two API endpoints available, one for all reports in JSON and one for reports by date in JSON or CSV. To retrieve all reports in JSON, use /reports.json Reports by date are at /YYYY-MM-DD/YYYY-MM-DD/reports.csv, .json, or .transit, where YYYY-MM-DD are the start and end dates requested.
